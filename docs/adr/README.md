## ADR Format

Each ADR follows this structure:

1. **Title** - Descriptive name of the decision
2. **Status** - Proposed, Accepted, Deprecated, Superseded
3. **Context** - Forces at play and the problem being solved
4. **Decision** - The decision that was made
5. **Consequences** - Both positive and negative
6. **Alternatives** - Options considered and why rejected
7. **References** - Supporting documentation
